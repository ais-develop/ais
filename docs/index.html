会社思想

近年、経済界・産業界・そしてスポーツ界までも、「グローバル化」と「ネットワーク化」によって大きな変貌を遂げてきました。 そして、その大きな波は一般家庭にも及んでいます。ＩＴ産業の中に生き続ける私たちは、常に新しい技術と向かい合わなければなりません。

私たちは、変化に対応し続けるたびに技術以外に、何が必要なのか、何を求められているのかを考え続けてきました。やがて、その問いを解くために一つの言葉を発想しました。

Accordance - 調和

すべての調和を考えたシステム、そして人創りを皆様と共に考えていきたいと思っております。
